¿Qué es CSS y para qué sirve?

CSS es un lenguaje de estilo que sirve para poder dar una forma visual / estilo a las paginas WEB, haciendo que visualmente estas se vean mejor. Con CSS se puede solucionar el problema de tamaños, localizaciones y muchos otros más de una forma muy facil y comoda.





Formas de aplicar CSS en HTML

Hay tres formas de aplicar un CSS a nuestro HTML

La primera forma es el inline, que es poner directamente en la linea que quieras modificar el estilo que quieras, es decir, si creas un "Div", en la misma linea en la que tienes el div puedes ponerle el tamaño, color, bordes, etc...

La siguiente forma es el interline, este se crea una etiqueta en la cabecera la cual se llama "Style", aqui deberás de primero de todo darle un nombre a las etiquetas que quieras modificar, poniendole dentro de la linea "class", una vez tengas puesto el nombre de class, deberas de ir a la cabecera donde style y poner el nombre que hayas puesto y "{}" para poner tu estilo.

Por ultimo tenemos el externo, que se basa en linkear un archivo css a nuestro HTML y en el archivo vacio con CSS podremos modificar las clases del archivo HTML.
