CSS margin

Establece el margen de los laterales, estos margenes se pueden especificar uno a una si queremos margen arriba, abajo, derecha o izquierda, pero si ponemos margin a secas se pondra el mismo margen en los 4 lados.

CSS padding

Establece el área de relleno, aquí tambien se pueden especificar entre arriba, abajo, izquierda y derecha, si ponemos padding a secas se pondrá el mismo relleno en todos los lados.

CSS border

Establece los bordes de un elementos, aquí tambien podemos especificar que lado queremos y si ponemos border a secas tambien se pondrá en los 4 lados.

CSS width and height

Width establece el ancho de un elemento mientras que Height establece el alto, estas dos caracteristicas se pueden poner con diferentes unidades de medidas como por ejemplo pixeles, porcetajes, viewport para hacerlos responisve, etc.
